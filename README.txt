README
======

SPACE DRIFTS PORTABLE(TM) 1.0
------------------------------

Obrigado por instalar o Space Drifts Portable 1.0!

Esta pasta inclui os códigos-fonte, arquivos de imagens e áudio, base de dados e o executável.


Sistemas Operacionais que rodam o Space Drifts Portable
-----------------------------------------------

O Space Drifts é suportado por Windows XP, Vista, 7, 8 e 8.1.



Conteúdo
--------

Incluem-se nesta versão os seguintes arquivos:

data\allegro-5.0.10-monolith-mt.dll - uma das bibliotecas da Allegro
data\resource.rc - resource para alteração de ícone do jogo
data\icone.ico - arquivo de ícone
data\saves.imd - arquivo de ranking
data\drifts.depend - arquivo de dependência do Code::Blocks
data\drifts.layout - arquivo de layout do Code::Blocks
data\drifts.cbp - arquivo de projeto do Code::Blocks
data\drifts.exe - executável do jogo
data\desenha_tela_ajustes.cpp - código C++ do método de desenho da tela de ajustes
data\desenha_tela_gameover.cpp - código C++ do método de desenho da tela de game over
data\desenha_tela_inicial.cpp - código C++ do método de desenho da tela inicial
data\desenha_tela_pre_ranking.cpp - código C++ do método de desenho da tela de gravação de pontuação
data\metodos_background.cpp - código C++ do método de movimentação do background
data\metodos_bolas.cpp - código C++ do método de movimentação das bolas verdes
data\metodos_bonus.cpp - código C++ do método de movimentação das bolas azuis
data\metodos_cursor_bola.cpp - código C++ do método de movimentação da bola amarela (cursor)
data\metodos_inimigos.cpp - código C++ do método de movimentação das bolas rosas
data\metodos_ranking.cpp - código C++ do método de manipulação do ranking
data\main.cpp - arquivo C++ central que contém o loop principal do jogo
data\audio\gameplay.ogg - trilha sonora das telas iniciais
data\audio\trilha_sonora.ogg - trilha sonora do jogo
data\audio\som_gameover.ogg - efeito sonoro da tela de game over
data\audio\som_bola.wav - efeito sonoro da colisão com as bolas verdes
data\audio\som_bonus.wav - efeito sonoro da colisão com as bolas azuis
data\audio\som_inimigo.wav - efeito sonoro da colisão com as bolas rosas
data\audio\som_botao.wav - efeito sonoro do clique no botão
data\bin\Debug\... - arquivos bin do projeto
data\obj\Debug\... - arquivos .o do projeto
data\fontes\Stopd.ttf - arquivo da fonte do jogo (Stopd)
data\imagens\game_over\... - imagens da tela de game over
data\imagens\jogo\... - imagens da tela de jogo
data\imagens\tela_ajustes\... - imagens da tela de ajustes
data\imagens\tela_inicial\... - imagens da tela inicial
data\imagens\tela_pre_ranking\... - imagens da tela pré ranking
data\imagens\tela_ranking\... - imagens da tela de ranking
